== README

LES Film Festival (LES FF) is a web app designed and executed in three days with a team of four using Agile strategies. Preview movies, leave reviews or vote for your favorites! LES FF demonstrates command of Ruby on Rails, JavaScript and its libraries, HTM and CSS.

* Ruby version
ruby 2.2.3

* Deployment instructions

  If you'd like to deploy the site, please download, migrate the database and run from local host.
